# **App Name**: InfinitTech Innovations

## Core Features:

- IT Services Showcase: A detailed presentation of the company's IT service offerings.
- Startup Pitch Page: A dedicated page with a compelling pitch for the internal startup to attract grants/funding. Features key information like the problem, solution, team, and financial projections.
- Grant eligibility AI tool: An AI powered eligibility tool for suggesting grants for the company.
- Investor Inquiry Form: A contact form designed for investors and potential partners to easily reach out and express their interest. Captures essential contact information and message details.
- About Us Section: A comprehensive section detailing the company's history, mission, vision, and team members, enhancing credibility and building trust.

## Style Guidelines:

- Primary color: Clean, professional blue (#468B97) to convey trust and stability.
- Background color: Light, desaturated off-white (#F5F5F5) for a clean and modern feel.
- Accent color: Muted green (#77AC97) to add a touch of innovation and growth, complementing the blue tones.
- Font pairing: 'Inter' for body text (sans-serif, neutral look) and 'Space Grotesk' for headlines (sans-serif, techy feel).
- Use a set of consistent, minimalist icons to represent IT services, startup goals, and company values, enhancing user experience.
- Employ a grid-based layout with clear sectioning to provide a structured, user-friendly browsing experience.
- Subtle animations and transitions to add interactivity and provide user feedback, enhancing engagement and creating a dynamic interface.
